# Careful with AI

**A Learn with Kaan book**  
*Learn AI Without the Pressure*

Slogan: **You can be careful and still begin.**

## Format
Short illustrated teaching book (~32–64 pages). Big type. Mascot pop-outs. One idea per spread. Not a dense textbook.

## Learning loop (every “Try this”)
1. **See** — what we’re doing today  
2. **Try** — one small prompt / action  
3. **Check** — does this make sense? would you send it?  
4. **Celebrate** — you stayed in charge
